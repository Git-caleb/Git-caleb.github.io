Approach

In the problem description, the term anagrams appears, which may be unfamiliar to non-native English speakers.An anagram is a word or phrase formed by rearranging the letters of a different word or phrase, using all the original letters exactly once.So the problem asks us to group strings that have the same number of letters together.
To solve this problem, we can use two common methods:

  • Sorting method: Sort the characters of two strings. If the sorted result is the same, the two strings are anagrams.
  • Counting method: Count the frequency of each letter in both strings. If the frequency counts for all letters match exactly, the two strings are anagrams.

I chose the first approach: I sort each string and use the sorted string as a key in a dictionary. If the key already exists in the dictionary, I append the original string to the corresponding value list.

Complexity Analysis
